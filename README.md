# Matify AI ‚Äî Reimagining math solving through drawing and AI üß†‚úèÔ∏è

Matify AI is a Next.js (v14) web app that lets users draw math problems on a canvas and get precise LaTeX solutions via Google GenAI (Gemini). It combines an interactive Excalidraw-powered whiteboard with rich text editing, user auth, and Convex as the backend datastore.

Key features
- Draw math problems and get immediate LaTeX solutions using Google Gemini.
- Excalidraw whiteboard for freeform math notation and diagrams.
- EditorJS-powered document editor for notes and structured content.
- Auth via Kinde, persistent data via Convex, and server-side GenAI integration.

## Feature Overview

- Whiteboard (Excalidraw)
	- Users can draw math equations, geometry, and diagrams.
	- The canvas can be exported and is captured to a Base64 image which is sent to `/api/calculate`.

- AI OCR + Solver (/api/calculate)
	- This endpoint sends a base64 PNG to the Google GenAI (Gemini) model with a system prompt that enforces LaTeX-only output.
	- The server returns a LaTeX string back to the client to render via `react-latex-next`.

- Documents (EditorJS)
	- EditorJS provides a structured document editor that saves content to Convex.

- Auth & Users
	- Kinde provides login/register flow.
	- On dashboard access, the app ensures a user record exists in Convex and creates one if necessary.
